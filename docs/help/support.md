## Other ways to help

If a financial donation isn't in the cards right now, here are
some alternatives that genuinely make a difference:

- **File a bug report** on GitHub when something doesn't work
  right. Specific reproduction steps + your Lyra version (shown
  in the title bar / About dialog) help fix things much faster.
- **Tell another HL2 owner** about Lyra. Word of mouth is how
  good ham software finds its audience.
- **Submit a pull request** if you're code-comfortable — Lyra is
  Python (PySide6 + NumPy + SciPy) and contributions are welcome.
  The codebase is documented inline; start with the issue tracker
  to see what's wanted.
- **73 from a ham QSL card** in the post is a real morale booster.
